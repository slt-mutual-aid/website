# South Lake Tahoe Mutual Aid

A static website built with Astro for South Lake Tahoe Mutual Aid.

## Development

```bash
# Install dependencies
yarn install

# Start development server
yarn dev

# Build for production
yarn build

# Preview production build
yarn preview
```
